## 조건부 렌더링
* 특정 조건에 따라 다른 결과물을 렌더링 한다.
-> 조건절 느낌인듯 (true/false)
```javascript
import React from 'react';

function Hello({ color, name, isSpecial }) {
  return (
    <div style={{ color }}>
      { isSpecial ? <b>*</b> : null } ⭐️null, false, undefined면 아무것도 안나타남!
      안녕하세요 {name}
    </div>
  );
}

Hello.defaultProps = {
  name: '이름없음'
}

export default Hello;
```

### && 연산자
```isSpecial && <b>*</b>``` 
* 위의 조건 값이 false면 false, true이면 * 를 나타낸다.
