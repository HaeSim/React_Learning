## BABEL
> - JavaScript 문법을 확장해주는 도구
> - 지원되지 않는 최신 문법/실험적 문법 등을 정식 JavaScript 형태로 변환해줌


## 기본적인 규칙(문법)
> - 태그는 꼭 닫혀있어야 함
> - 두개 이상의 태그는 하나의 태그로 감싸져있어야 함
> - JavaScript 값을 보여줘야 할 경우 {}으로 감싸기
```javascript
    <div style={style}>{name}</div>
```
> - 주석은 {/* 내용 */} 형태....(뭔가 복잡) 
```javascript
      {/* 주석은 화면에 보이지 않음 */}
      /* 중괄호로 감싸지 않으면 화면에 보임 */
```
> - 열리는 태그 내부에서느 // 형태로도 작성 가능
```javascript
      <Hello 
        // 이것도 주석
       />
```
